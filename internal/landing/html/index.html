Discovery Service

This is a cluster discovery service that enables nodes to exchange membership information and endpoints. All data is stored encrypted and expires after 30 minutes.

gRPC Service

Endpoint: :3000

The following gRPC methods are available:

  • Hello - Initial handshake with IP detection
  • AffiliateUpdate - Create or update affiliate records
  • AffiliateDelete - Remove affiliates
  • List - Retrieve all cluster affiliates
  • Watch - Stream real-time affiliate changes

Inspect Cluster

Rate Limits

Requests are rate-limited to 15 requests per second per IP address, with a burst allowance of 60 requests.

Limits

  • Maximum 100 affiliates per cluster
  • Maximum 20 endpoints per affiliate
  • Affiliate data expires after 30 minutes (configurable via TTL)
